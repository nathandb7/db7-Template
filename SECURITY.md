# Security Policy

db7 Template is a static HTML template. It does not include a backend, database, authentication system or server-side runtime.

## Supported Versions

Security reports are accepted for the current public version of the project.

## Reporting a Vulnerability

Please report security concerns privately to the maintainer.

Maintainer: Nathan de Barros  
Website: https://nathandebarros.com

Please include:

- A clear description of the issue.
- Steps to reproduce it.
- Affected file or feature.
- Potential impact.
- Suggested fix, if available.

## Scope

Relevant reports may include:

- Unsafe JavaScript behavior.
- Broken external links that create security risk.
- HTML injection risks in reusable examples.
- Supply chain issues in development dependencies.

Out of scope:

- Server-side vulnerabilities for deployments not included in this repository.
- Issues caused by custom user modifications.
- Third-party hosting configuration mistakes.
