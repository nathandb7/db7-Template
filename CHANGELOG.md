# Changelog

All notable changes to db7 Template will be documented in this file.

## 1.0.0 - 2026-07-16

### Added

- Initial open source release.
- 16 static HTML pages.
- 4 home page variants.
- Components library with reusable blocks.
- Copy code buttons for component snippets.
- Dark premium agency and portfolio visual identity.
- Responsive navigation with mega menu.
- Scroll reveal animation system.
- Animated counters.
- Local static development server.
- Mobile QA script using Playwright.
- Desktop and mobile screenshots in `docs/`.
- MIT License.
- Open source community documentation.

### Verified

- `npm run build`
- `npm run check`
- `npm run qa:mobile`
