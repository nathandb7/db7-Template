# Code of Conduct

db7 Template is an open source project. Participation should be respectful, constructive and welcoming.

## Expected Behavior

- Be respectful of different experience levels.
- Give feedback that is specific and useful.
- Assume good intent.
- Keep discussions focused on the project.
- Credit work and ideas where appropriate.

## Unacceptable Behavior

- Harassment, insults or personal attacks.
- Discriminatory language or behavior.
- Publishing private information without permission.
- Spam or repeated off-topic comments.
- Deliberately disruptive behavior.

## Reporting

If you see behavior that does not fit this code of conduct, please open a private report with the project maintainer.

Maintainer: Nathan de Barros  
Website: https://nathandebarros.com

Reports will be reviewed with care and handled as privately as possible.
