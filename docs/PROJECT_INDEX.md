# Project Index

A quick map for humans and AI assistants.

## Project

- Name: db7 Template
- Description: A premium HTML template for agencies, portfolios and service businesses.
- Author: Nathan de Barros
- License: MIT

## Source

- `src/*.html` - editable HTML pages.
- `src/components.html` - component library.
- `src/assets/css/style.css` - main stylesheet.
- `src/assets/css/plugins.css` - plugin placeholder stylesheet.
- `src/assets/js/main.js` - main interactions.
- `src/assets/js/modules/` - motion modules.
- `src/assets/images/` - image assets.
- `src/assets/video/` - video assets.

## Scripts

- `scripts/generate-pages.js` - generates HTML pages.
- `scripts/build.js` - copies `src/` to `dist/`.
- `scripts/clean.js` - removes `dist/`.
- `scripts/check-static.js` - checks local references.
- `scripts/dev-server.js` - local static server.
- `scripts/mobile-qa.js` - mobile QA screenshots and checks.
