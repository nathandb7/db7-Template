# db7 Template

A premium HTML template for agencies, portfolios and service businesses.

The Obry demo was used only as a visual and structural reference. No Obry HTML, CSS, JavaScript, text, images, or assets were copied into this project.

## Identity

- Brand: db7 Template
- Style: dark premium, editorial, high contrast, portfolio ready
- Primary accent: electric lime
- Secondary accent: cyan
- Detail accent: orange
